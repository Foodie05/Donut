# Donut Workspace

This repository is now organized as a small monorepo:

- `donut_app/`: the existing Flutter application.
- `donut_backend/`: a newly initialized Dart Frog backend service.

## App

```bash
cd donut_app
flutter pub get
flutter run
```

## Backend

```bash
cd donut_backend
dart pub get
dart_frog dev
```
